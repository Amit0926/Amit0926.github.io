## Features

- Dark-mode first futuristic UI with soft glow orbs and glassy cards.
- Animated AC logo badge with orbit effect.
- Sidebar layout with profile, navigation, and theme toggle.
- Sections: Overview, Skills Matrix, Experience, Education, Tools, Contact.
- Scrolling skill belt animation generated via JavaScript.
- Static contact form (ready to be wired to a backend or service).
- Responsive layout for desktop, tablet, and mobile.

## How to Use

1. Extract the ZIP.
2. Open `index.html` in any modern browser, or host the folder on:
   - GitHub Pages
   - Netlify
   - Vercel
   - Any basic shared hosting

You can edit text directly inside `index.html`, update colors and layout in `style.css`,
and adjust animations or skills in `script.js`.
